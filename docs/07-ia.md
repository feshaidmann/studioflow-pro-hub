# 07 · Inteligência Artificial

Toda IA passa pelo **Lovable AI Gateway** com a chave `LOVABLE_API_KEY` (sem chave do usuário, sem custo direto para o artista durante o beta).

## Modelos em uso

| Modelo | Uso principal |
|--------|---------------|
| `google/gemini-2.5-flash` | Chats curtos, classificação, geração de tarefas |
| `google/gemini-2.5-pro` | Análises complexas e revisões longas |
| `google/gemini-3-flash-preview` | Assistentes contextuais (dashboard, projeto, edital) |
| `google/gemini-3.1-flash-image-preview` | Geração de imagens em Direção Visual |

A escolha do modelo é feita por função e ajustada conforme custo/latência observados em `ai_invocations`.

## Quotas fair-use

`ai_usage` mantém contadores diário/semanal por usuário e função. Limites padrão: **20/dia** e **80/semana**. Quando excedido, a função retorna `rate_limit` e o frontend abre `RateLimitDialog` com horário de reset.

## Assistente IA "JamSession" (Dashboard)

- Persona: mentor de estúdio entusiasta, escopo estritamente técnico.
- Histórico em `ai_conversations` + `ai_messages`.
- Contexto injetado: projetos ativos, tarefas pendentes, finanças (apenas do próprio usuário).
- Chip "Dúvida técnica" muda persona para engenheiro de áudio.

## Assistente IA do Projeto

`project-ai-assistant` recebe stage, equipe, finanças e tarefas do projeto e devolve sugestões priorizadas. Componente `ProjectAISheet`.

## Assistente IA de Editais

`edital-ai-assistant` (`useEditalAI`) cobre as ações: `generate_memorial`, `adapt_language`, `review_budget`, `generate_checklist`, `suggest_project_fit`.

## DNA Musical

- Extração local via Web Audio API + `acousticMatch.worker.ts`.
- Classificador de gênero com calibração por usuário (`genre_mismatch_feedback`).
- Pipeline de referências: `music_reference_tracks` → benchmarks + exemplos no prompt.
- Persona técnica (engenheiro) com summary acessível.
- Tarefas geradas levam o prefixo `[DNA]` para rastrear origem.
- `enrich-neighbor-context` adiciona similaridade externa (Deezer/MB/LB) com cache 30 dias.

## Direção Visual

`generate-visual-direction` produz 6 variações via Gemini Image Preview, faz upload no bucket `creative-assets` e devolve URLs públicas. Briefings podem ser compartilhados publicamente via `/briefing/share/:token`.

## Padrão de UI para respostas de IA

Use sempre `AIMarkdownContent` para renderizar markdown produzido pelo modelo — garante tipografia consistente, spoilers de fontes e tratamento de blocos de código.
